# Back-End Development - Songs Microservice

This repository contains the Songs microservice, which manages the band's discography and track listings.

## Environment Setup
- **Created from template:** Yes
- **Successful execution of bin/setup.sh:** Yes
- **Python Version:** 3.9.x
- **Virtual Environment Name:** backend-songs-venv

## API Endpoints
- `GET /health`: Service health check
- `GET /song`: Retrieve all songs
- `PUT /song/<id>`: Update song details
- `DELETE /song/<id>`: Remove a song
